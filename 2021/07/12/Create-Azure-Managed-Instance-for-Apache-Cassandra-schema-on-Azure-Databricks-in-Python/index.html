本記事では、当該 Cassandra クラスターに対して、 Azure Databricks の Notebook 上で Python を使ってスキーマを作成する方法について解説します。スキーマの作成に成功することをゴールとします。

スキーマ情報は Azure Open Datasets で提供されている NOAA Global Forecast System のものを利用します。
